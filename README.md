# Todo Application

A full-stack Todo Application built using React, Node.js, Express, and a JSON file as the data store.

## Tech Stack

### Frontend
- React
- Axios
- CSS

### Backend
- Node.js
- Express.js
- UUID
- JSON File Storage

## Features

- Add Todo
- View All Todos
- Search Todos
- Filter Todos
- Mark Todo as Completed
- Mark Todo as Pending
- Edit Todo
- Delete Todo
- Responsive UI

## API Endpoints

| Method | Endpoint | Description |
|---------|-----------|-------------|
| GET | /todos | Get all todos |
| GET | /todos/:id | Get todo by id |
| POST | /todos | Create todo |
| PUT | /todos/:id | Update todo |
| DELETE | /todos/:id | Delete todo |

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend runs on

```
http://localhost:5000
```

Frontend runs on

```
http://localhost:5173
```
