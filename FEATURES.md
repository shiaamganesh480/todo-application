# Features and Functionalities

## Add Todo

Users can create a new todo by entering a title and clicking **Add Todo**.

---

## View Todos

Displays all todos fetched from the backend.

---

## Search Todo

Users can search todos instantly using the search box.

---

## Filter Todos

Three filters are available:

- All
- Pending
- Completed

---

## Update Status

Users can mark a todo as:

- Completed
- Pending

---

## Edit Todo

Users can edit the title of an existing todo.

---

## Delete Todo

Users can permanently delete a todo.

---

## Statistics

Displays:

- Total Todos
- Completed Todos
- Pending Todos

---

## Backend APIs

- GET /todos
- GET /todos/:id
- POST /todos
- PUT /todos/:id
- DELETE /todos/:id
